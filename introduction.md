# Introduction

ArtistDB is the open database of SFW/NSFW artists.

## Feature

- Quickly discovered new artists all around the world
- No AI artists
- Privacy friendly 
- Transparent & Verified data with human moderation

## Why?

AI are ruining the artists space and bad social media algorithm that hide many talented artists and user from seeing the art. ArtistDB will solve the problem like easy to use and info base site that help us discovered many of the artists all around the world. The site will only contain real and something that made by a human and not AI. All of the data will be manually reviewed by human too and give a verified badge if the artist confirm the data to be accurate.

## Will ArtistDB protect all of the data against AI?

We rely on [Cloudflare's AI Crawl Control](https://developers.cloudflare.com/ai-crawl-control/) to protect the website against AI scraper but despite all of that, we would not guaranteed 100% that it will block all of the bots because how easy can people bypass all of the Cloudflare and other DDOS services. 

## Does this project use AI?

85% Human, 15% AI

ArtistDB is still being build by ourselves but we still use AI to do some prototype stuff and an aid tools for helping us plan and build the project. We cannot use AI to generate the entire codebase because it would not follow the direction that we wanted and always give us shitty ass code and do not get me started on debugging the code that AI generated (Wrong API, do this but does different, etc.)

As much we hate AI generative works, using them for aids in development and planning are good in someway. But we still have to be careful with the AI and make sure it's not hurting the project.

We mostly use for frontend generation because building frontend suck but the rest just us coding the whole thing.
