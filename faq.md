# Frequently Asked Questions

## General

### What is ArtistDB?

ArtistDB is the open database of SFW/NSFW artists. It is an open-source platform enabling artists to showcase their work and connect with audiences created by [NYT92org](https://github.com/nyt92org).

### Does ArtistDB contain AI artists?

No, ArtistDB does not contain AI artists because ArtistDB is 100% clean and only allows human creators to be added to the database.

### Can I modify the data of the artist profile?

Yes, you can modify the data of the artist profile if you are the artist that has some of the data missing, incorrect, or not up-to-date information. 

However, you cannot use our data to modify it since all of our data is licensed under [Creative Commons Attribution-NoDerivatives 4.0 International License (CC BY-ND 4.0)](https://creativecommons.org/licenses/by-nd/4.0/). Please read more on [Terms of Service](/terms-of-services#data-content)

### Can I self-host ArtistDB?

Yes, you can self-host ArtistDB website and API. If you want to modify the source code, you have to release your source code under the same license as ArtistDB (GPL-3.0).

## Submission

### Can I submit myself to ArtistDB as an artist?

We welcome all artists around the world to join ArtistDB and have their own profile to showcase their work. Please refer to [Submission Guidelines](/submission) for more information.

### Can I submit other artist to ArtistDB?

Yes, you can submit other artist to ArtistDB (if possible request their permission first). Please refer to [Submission Guidelines](/submission) for more information.

### How can I get verified?

You can get verified by submitting your artist profile to ArtistDB and having it approved by our moderators. Please refer to [Getting Verified](/verified-by-artistdb#how-to-get-verified) for more information.

### I saw my profile on there but I didn't submit it myself, what should I do?

If you see your profile on ArtistDB but you didn't submit it yourself as an artist, you will be ok unless there is something wrong with the profile like missing, incorrect, or not up-to-date information. Please contact us at [support[at]artistdb[dot]org](mailto:support@artistdb.org) and we will discuss with you about the issue.

### I want to remove my profile from ArtistDB, what should I do?

We do not randomly remove the artist profile even though we find the artist using AI or doing something against the rules. We will only remove the artist profile if the artist themselves request it by providing proof that they are the artist and the valid reason for the removal.

If you want to remove your profile from ArtistDB, please contact us at [support[at]artistdb[dot]org](mailto:support@artistdb.org). You will need provide proof that you are the artist with the valid reason for the removal. We will review it and remove your profile from the database if it is valid.

## API

### Can I use ArtistDB API for my own project?

Yes, you can use ArtistDB API for your own project but you need to follow the rules in the [API Documentation](/api) and read our [Terms of Service](/tos).

### Can I self-host ArtistDB API?

Yes, you can self-host ArtistDB API for your own database but if you need a mirror of our database, please contact us at [support[at]artistdb[dot]org](mailto:support@artistdb.org). **Only for commercial use**

### Can I request a higher rate limit?

:::warning
You can only request from 60 up to 150 requests per minute right now while the default rate limit is 60 requests per minute.
:::

Yes, you can request a higher rate limit by contacting us at [support[at]artistdb[dot]org](mailto:support@artistdb.org) also you will need provide the reason for the request.
