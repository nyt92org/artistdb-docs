## Guidelines

1. **Artists Only**: This database is exclusively for artists that DRAW.
2. **Authenticity**:
    - Submit only real, existing artists
    - All required artist data fields must be completed when submitting.
3. **AI Art Policy**:
    - AI artists are not eligible for submission
    - Artists who have recently incorporated AI tools may be submitted but will be reviewed and potentially marked in a **Artist Notices** if necessary
4. **For Group/Collective Submissions**:
    - For artist groups or duos, submit each member as a separate entry
    - Include clear references to their collaborative work on description or aliases
5. **Artist Aliases**:
    - Submit one primary entry per artist
    - Include all known aliases in the artist's data
    - Do not create separate entries if the artist has different aliases
6. **Social Media Requirements**:
    - At least one or more active social media link with 2-4 arts on there
    - Links must be current and accessible
    - Links must have either user ID or username or handle in the URL
    - Links cannot be duplicate or redirect to the same URL  
        (twitter.com/artist1 ✅ - twitter.com/intent/user?user_id=1234567890 ❌)
7. **Duplicate**:
    - Check existing entries before submitting
    - Artist with the same aliases or name cannot be submitted (Rule 5, Point 3)
8. **Content Guidelines**:
    - If possible, ask permission before submitting artist when showing their showcase media
    - NSFW tags/content must be properly flagged
    - Respect artist privacy and copyright
    - Do not submit private contact information (addresses, phone numbers, emails)
    - Only include information that is publicly available or explicitly authorized or have been confirmed by the artist themselves.
